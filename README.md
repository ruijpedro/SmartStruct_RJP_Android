# SmartStruct_RJP Core 100

Ferramenta não comercial para uso académico e apoio técnico no âmbito da Infraestruturas de Portugal.

## Inclui
- React + Vite + TypeScript
- WebApp compilável
- Base Capacitor/Android
- GitHub Actions para WebApp e APK
- Arquitetura modular com Core, UI, estruturas, geotecnia, ferrovia e biblioteca técnica
- Módulos ainda em base estrutural, preparados para implementação funcional faseada

## Nota
Não substitui projeto, parecer técnico, verificação regulamentar completa ou responsabilidade de técnico habilitado.
